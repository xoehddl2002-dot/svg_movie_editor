> **"AI가 실제로 생산성과 비용 절감에 기여할 수 있는가?"**

이 프로젝트는 **두 가지 측면**에서 이 질문에 답합니다:

## 🤖 AI 바이브코딩 (Vibe Coding)

본 프로젝트의 주요 기능 구현은 **AI 코드 생성 도구(Gemini, Copilot 등)를 적극 활용한 바이브코딩** 방식으로 진행되었습니다.

### 바이브코딩이란?
개발자가 자연어로 의도를 전달하면, AI가 코드를 생성하고, 개발자는 이를 검토·조합하여 빠르게 결과물을 만들어내는 **AI 협업 개발 방식**입니다.

### 적용 결과

| 항목 | 효과 |
|------|------|
| **개발 속도** | 짧은 기간 내 프로토타입 구현 |
| **코드 품질** | AI 제안 기반 구조 설계로 일관성 확보 |
| **전문 영역 접근** | SVG 파싱, FFmpeg 연동 등 전문 영역의 러닝커브 절감 |
| **반복 작업 자동화** | 보일러플레이트·유사 패턴 코드 자동 생성 |

> 💡 특히 **SVG 렌더링**, **타임라인 로직**, **마스크 편집**, **영상 내보내기** 등 복잡한 기능의 초안을 AI가 빠르게 생성하면서 전체 개발 사이클이 크게 가속되었습니다.

## 💰 SVG × 텍스트 AI 모델 = 비용 혁신

이 프로젝트의 핵심 인사이트:

### SVG는 "이미지처럼 보이지만, 실제로는 코드"입니다

```xml
<!-- 이것은 이미지가 아닌 텍스트 코드입니다 -->
<svg viewBox="0 0 200 200">
  <circle cx="100" cy="100" r="80" fill="#4F46E5"/>
  <text x="100" y="110" text-anchor="middle" fill="white" font-size="24">
    Hello SVG
  </text>
</svg>
```

이 특성을 활용하면, 비싼 이미지/영상 생성 AI 대신 **저렴한 텍스트 생성 AI**로 시각 콘텐츠를 만들 수 있습니다.

### 비용 비교

| 구분 | 모델 유형 | 상대 비용 | 출력 |
|------|-----------|-----------|------|
| **이미지 생성** | 이미지 모델 (DALL·E, Midjourney 등) | 💰💰💰 높음 | 래스터 이미지 (PNG/JPG) |
| **영상 생성** | 영상 모델 (Sora, Runway 등) | 💰💰💰💰💰 매우 높음 | MP4 영상 |
| **SVG 생성** | 텍스트 모델 (GPT, Gemini 등) | 💰 낮음 | 벡터 그래픽 (SVG) |

### SVG 접근 방식의 장점

- ✅ **비용 효율**: 텍스트 모델 API 비용은 이미지·영상 모델 대비 **수십 배 저렴**
- ✅ **해상도 독립**: 벡터 기반이므로 어떤 해상도에서도 선명한 출력
- ✅ **편집 가능**: 생성 후에도 개별 요소 수정·애니메이션 적용 가능
- ✅ **경량**: 파일 크기가 작아 웹 성능에 유리
- ✅ **자동화 가능**: 코드 기반이므로 대량 생산·템플릿화 용이

> 📊 **결론**: SVG + 텍스트 AI 모델 조합은 **시각 콘텐츠 생성의 비용 장벽을 획기적으로 낮추는** 새로운 접근법입니다.

## 🎯 개발 목표 및 달성 현황

### 1차 목표: 기존 규격 통합 편집기 구축 (완료)
기존 **사장님 TV SVG 규격**을 영상으로 변환할 수 있도록 편집기를 확장하고 통합하는 작업을 성공적으로 완료했습니다.

### 2차 목표: AI 기반 콘텐츠 생성 파이프라인 (완료)
AI 모델을 적극 활용하여 사용자가 더욱 손쉽게 자신만의 템플릿을 생성할 수 있는 기능을 추가했습니다:
- **Text to SVG Template**: 텍스트 프롬프트를 입력하여 사용자의 의도나 컨셉에 맞는 새로운 SVG 템플릿을 자동으로 생성하는 기능
- **Image to SVG Template**: 참조 이미지를 Base64로 인코딩한 뒤 텍스트 AI 모델(Vision)에 전달하여, 원본 이미지와 유사한 레이아웃/스타일의 SVG 템플릿을 추출해내는 기능
